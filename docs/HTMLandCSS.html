HTML & CSS



HTML

HTML stands for Hyper Text Markup Language, this is the language used to tell a web browser how you would like a page to look. The text of the webpage is included within HTML, however, other parts like images or videos are separate files with their unique URLs. When your browser sends an HTTP request it sends an individual request for each of URLs and it displays the corresponding content as it receives a response.

Using HTML you can define the formatting of the text, make it bolded, italics, underlined. You can also define the position of the content within the page (centered, aligned to the left, right, top or bottom). However, for the most part HTML is used to define the structure of the content within a website.


CSS

CSS stands for Cascading Style Sheet. It defines how the HTML elements will be displayed, in other words, just like HTML defines the structure of the content in a website, CSS defines the style of this content. One of the many advantages of CSS is that you could apply the same formatting to multiple elements of the HTML code within a single reference.
