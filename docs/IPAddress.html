The Internet - IP Addresses


All the devices connected to the internet have an address. An address on the internet is represented by a number that is unique to each device on the network. A computer’s address is called an IP address. When you send a message from one computer to another, you need its IP address and within the message you also need the sender's IP address so that the remote computer knows where to respond.


IPv4


In 1973, the Internet Protocol version 4 (IPv4) was designed. Like all digital data, an IPv4 address is represented in binary code. To be more precise, an IPv4 address is represented by 32 bits divided into four octets. The first octet usually represents the country of the network, followed by the region, subnetwork and then the device address. IPv4 provides for more than 4 billion unique addresses, however, due to the popularity of the internet; this amount of IP addresses won’t be enough.


IPv6


In order to solve this problem a new version of the Internet Protocol has been developed, IPv6. An IPv6 address uses 128 bits, which provides 340 undecillion unique addresses. Nonetheless, it will be a slow and multi-year transition from IPv4 to IPv6.
