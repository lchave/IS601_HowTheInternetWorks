IS 601 - How the Internet works?


How the Internet works is a website created by Luis Chavez Saenz, a graduate student from NJIT, as a required assignment for the IS 601:Web System Development course of the Spring 2021 semester. The content of the website briefly covers the following topics:

  • Packets & Routing
  • Domain Names & ICANN
  • IP Address
  • Domain Name System (DNS)
  • HTTP\HTTPS
  • HTML & CSS
  • W3C

The goal is to understand how each of these topics and definitions have made the internet such a success.
